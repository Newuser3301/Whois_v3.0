= Pentest Recon Tool
:toc:
:toclevels: 3
:sectnums:
:icons: font
:source-highlighter: rouge

Python/PyQt6 asosidagi WHOIS, DNS, passive OSINT, WAF/CDN fingerprint, Nmap port scan, web technology, CVE va TLS sertifikat tahlil GUI dasturi.

Ilova bitta domen bo'yicha rekognitsiya natijalarini alohida tablarda ko'rsatadi va JSON/TXT hisobot saqlash imkonini beradi.

IMPORTANT: Dasturdan faqat o'zingizga tegishli yoki tekshirishga yozma ruxsatingiz bor domenlar uchun foydalaning. Nmap port scan aktiv tekshiruv hisoblanadi.

== Asosiy Imkoniyatlar

[cols="1,3", options="header"]
|===
|Tab
|Vazifasi

|WHOIS MA'LUMOTLARI
|Domen ro'yxatdan o'tgan registrar, sana, status, DNSSEC, abuse contact va name serverlarni ko'rsatadi.

|DNS RECORDS
|A, AAAA, MX, TXT, NS, CNAME, SOA, CAA yozuvlarini tekshiradi. Alohida record yoki hammasini qayta tekshirish mumkin.

|SUBDOMAINS
|Subfinder mavjud bo'lsa undan, aks holda ko'p public passive manbalardan subdomain yig'adi.

|OSINT
|Faqat passive manbalardan email, URL, interesting URL, passive host va source statuslarni chiqaradi.

|WAF/FIREWALL
|DNS, IP range, HTTP header/cookie/server fingerprintlari orqali WAF/CDN provayderini taxmin qiladi.

|NMAP PORTS
|`python-nmap` orqali top 100 TCP portni tekshiradi va faqat ochiq portlarni ko'rsatadi.

|TECH/CVE
|`builtwith` va HTTP header fingerprintlari orqali texnologiyalarni topadi, `nvdlib` orqali mos CVE raqamlarini izlaydi.

|TLS
|`cryptography` va `pyOpenSSL` orqali TLS sertifikat, SAN, issuer, validity, cipher va protocol ma'lumotlarini chiqaradi.

|HUJUM VEKTORLARI
|DNSSEC, SPF, DMARC, Cloudflare/origin IP va boshqa indikatorlardan xavfli joylarni umumlashtiradi.

|RAW OUTPUT
|Barcha natijalarni raw JSON/text ko'rinishida ko'rsatadi.

|HISOBOT
|TXT yoki JSON hisobot saqlash va natijalarni clipboardga nusxalash imkonini beradi.
|===

== Talablar

=== Python

Python 3.11+ tavsiya qilinadi. Dastur sizdagi muhitda Python 3.15 bilan ishlatilgan.

=== Python kutubxonalari

Minimal kerakli kutubxonalar:

[source,powershell]
----
pip install PyQt6 python-whois dnspython python-nmap requests cryptography pyOpenSSL builtwith nvdlib Wappalyzer
----

NOTE: `Wappalyzer` paketi Playwright dependencylarini o'rnatishi mumkin. Hozirgi kod asosan `builtwith` va header fingerprintlardan foydalanadi; Wappalyzer keyingi kengaytirish uchun tayyor turadi.

=== Tashqi dasturlar

`python-nmap` ishlashi uchun tizimda `nmap.exe` o'rnatilgan va PATH orqali topiladigan bo'lishi kerak.

Windows uchun:

[source,text]
----
https://nmap.org/download.html
----

Tekshirish:

[source,powershell]
----
Get-Command nmap
----

Subdomain topish kuchliroq bo'lishi uchun optional:

[source,powershell]
----
subfinder -version
----

`subfinder` mavjud bo'lmasa ham dastur public passive manbalar bilan ishlayveradi.

== Tezlik va Asinxronlik

Recon worker `asyncio` va `ThreadPoolExecutor` orqali mustaqil ishlarni parallel bajaradi:

* WHOIS
* DNS
* passive subdomain
* TLS
* technology detection
* Nmap
* WAF/CDN fingerprint
* OSINT URL/email yig'ish
* CVE qidirish

Bitta modul xato bersa, butun scan yiqilmaydi. O'sha tabda status yoki qisqa xabar ko'rsatiladi.

== Passive Subdomain Manbalari

Quyidagi manbalar ishlatiladi:

* `subfinder`, agar tizimda o'rnatilgan bo'lsa
* `crt.sh`
* CertSpotter
* AlienVault OTX
* HackerTarget
* URLScan
* RapidDNS
* Wayback CDX
* BufferOver
* ThreatMiner
* AnubisDB

Public manbalar ba'zan `429`, `500`, `503`, DNS error yoki timeout qaytarishi mumkin. Bu normal holat. GUI `OSINT -> Sources` bo'limida buni `rate limited`, `timeout`, `service unavailable` kabi qisqa status bilan ko'rsatadi.

== WAF/FIREWALL Aniqlash

Aniqlash quyidagilarga asoslanadi:

* DNS markerlar
* Cloudflare IP range
* HTTP response headerlari
* Cookie/server fingerprintlar

Taniladigan provayderlar:

* Cloudflare
* Akamai
* AWS CloudFront / AWS WAF
* Fastly
* Imperva / Incapsula
* Sucuri
* F5 BIG-IP ASM
* Barracuda
* StackPath
* Azure Front Door / App Gateway
* Google Cloud Armor
* DDoS-Guard
* BunnyCDN

NOTE: Bu fingerprint taxminiy. WAF topilmasa, bu WAF yo'q degani emas.

== Nmap Port Scan

Nmap tab quyidagi argument bilan ishlaydi:

[source,text]
----
-Pn -T3 --top-ports 100 --open
----

Bu faqat top 100 TCP port ichidan ochiqlarini ko'rsatadi.

CAUTION: Nmap aktiv scan hisoblanadi. Faqat ruxsatli targetlarda ishlating.

== Technology va CVE

Technology detection:

* `builtwith.parse()`
* HTTP header fingerprintlari

CVE lookup:

* `nvdlib.searchCVE()`
* Har bir aniqlangan texnologiya bo'yicha cheklangan miqdorda CVE qidiriladi.

NOTE: NVD API rate-limit qilishi mumkin. Bunday holatda `TECH/CVE -> Sources` yoki `CVE Matches` statuslarida ko'rsatiladi.

== TLS Tahlili

TLS tab quyidagilarni chiqaradi:

* subject
* issuer
* serial number
* not before / not after
* signature hash
* SAN ro'yxati
* cipher name
* TLS protocol
* oddiy issue indikatorlari

Kutubxonalar:

* `cryptography`
* `pyOpenSSL`
* Python `ssl`

== Hisobotlar

`HISOBOT` tabida:

JSON export `last_results` obyektini yozadi, ya'ni barcha tab natijalari strukturali saqlanadi.

== Troubleshooting

=== GUI ochilmayapti

PyQt6 o'rnatilganini tekshiring:

[source,powershell]
----
python -c "import PyQt6; print('ok')"
----

=== Nmap ishlamayapti

Quyidagilarni tekshiring:

[source,powershell]
----
python -c "import nmap; print('python-nmap ok')"
Get-Command nmap
----

Agar `Get-Command nmap` natija bermasa, Nmap o'rnating va PATH ga qo'shing.

=== Passive source xatolari chiqyapti

`429`, `500`, `503`, `timeout` va DNS xatolari public OSINT servislarida odatiy. Dastur qolgan manbalar bilan davom etadi.

=== CVE natija chiqmayapti

Sabablar:

* NVD vaqtincha javob bermayapti
* rate-limit
* texnologiya nomi juda umumiy
* versiya aniqlanmagan

=== TLS error

Sabablar:

* 443 port yopiq
* SNI qo'llab-quvvatlanmaydi
* sertifikat zanjiri muammoli
* tarmoq/proxy xatosi

== Xavfsizlik va Mas'uliyat

Ushbu dastur xavfsizlik auditi, o'quv va ruxsatli rekognitsiya uchun mo'ljallangan.

Ruxsatsiz port scan, agressiv tekshiruv yoki uchinchi tomon infratuzilmasiga zarar yetkazish noqonuniy bo'lishi mumkin.

Tavsiya:

* faqat o'z domenlaringizda ishlating;
* mijoz domenida yozma ruxsat oling;
* Nmap scan ishlatishdan oldin scope va vaqt oynasini kelishib oling;
* public OSINT API rate-limitlarini hurmat qiling.

== Hozirgi Cheklovlar

* WAF fingerprint taxminiy.
* CVE qidiruv versiyasiz texnologiya nomiga asoslangan bo'lsa false-positive bo'lishi mumkin.
* Passive OSINT manbalar barqaror emas; rate-limit yoki timeout bo'lishi mumkin.
* Subdomain tab GUI tezligi uchun birinchi 1000 ta natijani ko'rsatadi; to'liq ro'yxat RAW/JSON hisobotda saqlanadi.
* Nmap faqat top 100 TCP portni tekshiradi.
